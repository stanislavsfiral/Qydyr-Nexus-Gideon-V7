**QYDYR NEXUS** — это технологический прорыв в области аппаратной безопасности (PHY-security). Система реализует концепцию «физического якоря доверия», обеспечивая контроль целостности данных до их попадания в цифровую среду.

## 💎 Суть технологии
В отличие от классических методов (TDR), работающих с импедансом, GIDEON v13.7 отслеживает **фазовый сдвиг (Δϕ)** в уникальной S-топологии. Это позволяет фиксировать факт физического вмешательства в линию передачи данных с точностью **AUC=0.998**.

### Ключевые особенности:
* **Топологический замок:** Геометрия S-переходов встроена в физический слой передачи (Hard IP).
* **Динамический PUF:** Каждая линия связи становится уникальной физической функцией (Physical Unclonable Function).
* **Крипто-якорь:** Интеграция с Ed25519 через аппаратные прерывания. Если физический слой нарушен — логическая подпись блокируется мгновенно.

## 🚀 Демонстрация (Web-Engine)
В данном репозитории представлена визуальная модель ядра GIDEON, реализованная на базе **Three.js**.

1. **512 Сегментов:** Модель витков Сфирали дискретизирована на 512 пикселей для визуализации когерентности сигнала.
2. **Phase Resonance:** В реальном времени отображается телеметрия фазового резонанса (Δϕ).
3. **Video-Mapping:** Видеопоток проецируется на топологию, демонстрируя работу «информационной жилы».
